# 调用外部包中的代码

访问 [pkg.go.dev](https://pkg.go.dev/), 搜索[quote](https://pkg.go.dev/search?q=quote)

可能遇到代理问题
```bash
go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOSUMDB=off
go clean -modcache
go mod tidy
```

```go
package main

import

(

    "fmt"

    "rsc.io/quote/v4"

)

func main() {

    fmt.Println("Hello, World!")

    fmt.Println(quote.Go())

    fmt.Println(quote.Hello())

}
```
